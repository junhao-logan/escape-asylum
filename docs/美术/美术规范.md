# 美术规范

> 美术负责人维护。本文件是美术风格的"单一事实来源"，所有美术产出对齐这里。

## 风格定位

- 整体风格：**SCP / 超现代收容所**。
- 收容所探索：俯视 2.5D 视角。
- 战斗场景：第三人称视角。

## 资源提交约定

- 概念图、参考图放本文件夹（`docs/美术/`）。
- 进入游戏的正式资源放 `godot-project/` 内对应目录。
- 大文件走 Git LFS（见根目录 `.gitattributes`）。

## MVP 阶段建议

Slice 阶段美术资源不足时，**先用占位素材（灰盒 / 免费素材）**把玩法跑通，
美术精力优先投在投资人最先看到的部分：主角、第一层场景、卡牌 UI。
