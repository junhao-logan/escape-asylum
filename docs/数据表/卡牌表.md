## 字段说明

- **序号**：卡牌唯一编号
- **名称**：卡牌名
- **类型**：主动 / 被动
- **效果**：具体效果描述
- **消耗**：能量消耗
- **流派 / 罪**：归属哪个流派（七宗罪），通用卡留空
- **状态**：草稿 / 待评审 / 已实现 / 已测试
