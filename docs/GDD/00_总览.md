## 一句话概念

玩家扮演收容所中代号"零号罪人"的异常实体，凭借七宗罪之一的能力，
通过卡牌战斗一层层击败看护人员与 BOSS，最终逃出收容所。

## 核心卖点

- **七宗罪构筑**：开局选择一种罪作为核心能力，决定整局的卡牌流派玩法。
- **时停打牌的爽感**：战斗中可释放"时缓"，从容出牌；目标是带给玩家持续、密集的正反馈。
- **肉鸽 + 局外成长**：每局地图随机；通过"收容报告/随身笔记本"积累跨局成长。
- **碎片化叙事**：七宗罪与七美德的对抗背景，通过特殊房间逐步揭示。

## 核心循环（玩家反复在做什么）

进入房间 → 战斗 / 触发房间事件 → 获得道具与卡牌 → 强化自身 →
进入下一房间 →（重复）→ 击败本层 BOSS → 进入下一层

详见 `01_核心循环.md`。
