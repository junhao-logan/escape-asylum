# 05 · 世界观与剧情

> 本篇整理自《游戏大方向概括》。世界观信息量大，但 MVP / Slice 阶段
> **只需要呈现一小部分**——重点是氛围和"七宗罪 vs 七美德"的钩子。

## 世界设定

- **星球**：雾霄星。
- **四个国家**：埃克索德（风暴之乡）、亚索斯（工业最高、犯罪率最高）、
  洛夫兰（律法森严的古国）、纳芬斯（自然类源体自留地）。
- **核心机构**：源体之庭（成立于秩序历 3027 年），下设源体阁、登记司、
  审判厅、研究处、收容科。主要故事发生在亚索斯的源体之庭。
- **美术风格**：SCP / 超现代。

## 核心概念

- 部分实体 / 概念会"实例化"。古代被奉为神。
- 能吸收概念能力的人称为**源体**。
- 机构通过从实体提取能力转移给人类来扩张势力。
- 收容所关押特异生物 / 囚犯，作为研究与能量来源。

## 主角

- 代号"**零号罪人**"，七宗罪的结合体。
- 不为人类但具有人类外表特征。
- 最早被发现、但从未被解析的收容物。
- 主角失忆。
- 开局从七宗罪中选一个作为能力。

## 七宗罪能力（开局七选一）

| 罪 | 能力方向 |
|---|---|
| 傲慢 | 战胜敌人后可不拿战利品，转为自创卡牌 / 饰品 |
| 嫉妒 | 敌人打出两张相同的牌时，自己获得 buff |
| 愤怒 | 每回合随机打牌，牌的效果更强 |
| 怠惰 | 每次打牌少于 X 张时加能量，能量上限提升 |
| 贪婪 | 牌数量达到一定程度获得 buff |
| 暴食 | 吞噬装备武器，吸收加攻击或化为一次性卡牌 |
| 色欲 | 攻击敌人时自己获得随机 buff |

## 七宗罪 vs 七美德

| 七美德 | 对应七宗罪 |
|---|---|
| 贞洁 Chastity | 色欲 Lust |
| 节制 Temperance | 暴食 Gluttony |
| 慷慨 Charity / Kindness | 贪婪 Greed |
| 勤勉 Diligence | 怠惰 Sloth |
| 忍耐 Patience | 愤怒 Wrath |
| 宽容 Charity | 嫉妒 Envy |
| 谦卑 Humility | 傲慢 Pride |

## 结局设计

- **假结局**：最终 BOSS 为七美德之一。
- **真结局**：七宗罪全部通关后解锁，最终 BOSS 为"至高美德"（七美德集合体）。
- 若七美德未全部战胜过，通关时会"剧情杀"——因至高美德无缺陷。

## 叙事方式

- 碎片化叙事。
- 特殊房间（收容间、观察室、医疗仓）触发剧情。
