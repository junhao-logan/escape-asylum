# 04 · 关卡与肉鸽

## 关卡结构

- 总共 **7 层**。
- 房间随机生成。
- 击败本层 BOSS 进入下一层；通关第 7 层视为逃出收容所。

## 房间类型

| 房间 | 内容 |
|---|---|
| 普通战斗房间 | 常规敌人战斗 |
| 精英怪房间 | 强敌，奖励更好 |
| 机关陷阱房间 | 环境机关挑战 |
| 资源房间 | 回复能量 / 血量 / 属性 / 道具 |
| 卡牌奖励房间 | 解锁新卡 / 替换卡 |
| 商店房间 | 用资源购买卡牌、道具 |
| 随机房间 | 内容随机 |
| 事件房间 | 剧情 / 抉择事件 |
| BOSS 房间 | 本层 BOSS |

## 特殊剧情房间

收容间、观察室、医疗仓等特殊房间承载碎片化叙事，详见 `05_世界观与剧情.md`。

## 肉鸽元素

- 每次开局地图结构随机。
- 跨局成长：随身笔记本 / 收容报告 / 开局饰品。
