# 02 · 卡牌系统

## 七罪槽位

玩家有 **7 个技能槽位**，分别对应七宗罪。每个槽位放一张卡牌，对应一个键位（暂定 1–7）。

## 卡牌类型

- **主动型**：玩家手动释放。
- **被动型**：满足条件自动释放。

## 卡牌获取方式

- 每关打完给卡包（参考《炉石传说》开卡包的局外成长感）。
- 卡牌奖励房间：解锁新卡 / 替换已有卡。
- 商店房间购买。
- 事件房间、特定流派专属。
- 主角能力衍生（如"傲慢"可自创卡牌，"暴食"可把装备化为一次性卡牌）。

【待定】卡池设计方向二选一或结合：
- 方案 A：所有流派共享一个大卡池。
- 方案 B：一个共享卡池 + 各流派专属卡池（构筑更复杂、更有 build 感）。

## 卡组构筑相关

- 时停打牌：进入时缓后出牌，手牌全部打完才能刷新弃牌堆。
- 局外成长：通过"随身笔记本 / 收容报告"积累跨局解锁。
