# 01 · 核心循环

## 目标

通过收集物资和探索地图，击败每一层的 BOSS，最终逃出收容所。

## 核心机制（玩家主要重复在干什么）

1. 进入一个房间
2. 战斗 / 触发房间事件
3. 收获道具（饰品、卡牌）
4. 强化自身
5. 进入下一个房间
6. 重复以上多次
7. 击败本层 BOSS，进入下一层

## 主要阻碍

- 敌人类型多，且大多为"机制怪"（每种敌人有独特机制，需要不同应对）。
- 战斗动静太大会吸引其他敌人。【待定：具体规则】
- 打的敌人波次越多，下一层越难。【待定：难度曲线如何量化】
